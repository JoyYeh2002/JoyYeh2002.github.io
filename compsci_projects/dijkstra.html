Dijkstra Path Finder (JAVA)

April 2021
EN.600.226 Data Structures

Objectives:

  • Implement the vertex, two-way edge, and labels of a sparse graph
  • Implement Dijkstra’s algorithm
  • Load a .txt file with the distances between every street in Baltimore
  • Find the shortest path between two points

    • Skills Applied:

  • Writing Java unit tests, priority queue

    • Comments:

    During this project, I learned from online about how lightnings implement Dijkstra’s algorithm when finding the smallest-resistance path through the air. A slow-motion video of a lightning strike looks astonishingly similar to the animation demos shown in class. It was a truly fascinating correspondence.
